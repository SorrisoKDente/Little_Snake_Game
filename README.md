Desenvolvido como projeto prático para a faculdade de Engenharia de Software, este jogo utiliza a biblioteca Pygame para explorar conceitos fundamentais de lógica de programação, como loops infinitos, detecção de colisão e manipulação de coordenadas em tempo real. O foco principal foi a aplicação de estruturas de controle e funções em Python, garantindo um código organizado e eficiente.
